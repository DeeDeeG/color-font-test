Instructions

Please double-click each font file in the main folder for this repo, and install each font.

See README.md for getting fonts that aren't included in the repo.

(Seeing a lot of magenta? That's the color all fallback glyphs will have, or glyphs that fail to render with their intended font and color. You might still need to obtain and/or install the fonts. Or maybe your browser doesn't support all four formats. Most don't. Or maybe you forgot to obtain/install the fonts, *AND* your browser doesn't support the format anyway.)

(If you see magenta on the <-- LEFT, you either did not (were not able to?) install that font, the font is broken, or your browser does not support that color font format. If you see magenta on the --> RIGHT, you either haven't placed the font file in the same folder as the test.html file, the font's filename doesn't match the CSS @font-face declaration, the font is broken, or your browser does not support that color font format.)

In any case, if the visual on the <-- LEFT matches the visual on the --> RIGHT, and both sides look how the font is supposed to look, e.g. they aren't both the fallback font, your browser supports that format of color font. If not, either the test is broken, the font is broken, or the browser does not fully support that format of color font.
